## 4. Observed content gaps (competitors/aggregators rank because RVCE pages are thin)
Aggregators (Shiksha, Careers360, getmyuni, Quora) currently capture IEM search traffic. **Gaps the official site should fill:** clear placement statistics page, a "Careers / what you can do" page, an FAQ page, faculty research profiles, alumni success stories, events log, downloadable syllabus.

## 5. On-page SEO recommendations
- Unique `<title>` + meta description per page with primary keyword (templates embedded in each built page).
- Semantic headings (one H1/page), descriptive alt text on every image.
- JSON-LD structured data: `CollegeOrUniversity` / `EducationalOrganization` + `FAQPage` (already scaffolded in the built site).
- Clean descriptive URLs (`/placements.html`, `/faculty.html`), internal linking between pages, mobile-first responsive, fast load (system fonts, minimal JS).
- Canonical tags; sitemap.xml + robots.txt; Open Graph/Twitter cards for shareability.

## Best-practice features observed on top IE department sites
1. **Clear program framing** — "what is IE", outcomes, and career destinations front-and-center.
2. **Faculty research portal** — searchable profiles with publications, labs, areas.
3. **Research/labs pages** with funded projects and centres.
4. **Placement/careers page** with statistics, recruiters, role maps.
5. **News & events** feed; student-chapter highlights.
6. **Prospective-student CTAs** — admissions, brochure download, contact.
7. **Alumni success stories** with photos and current roles.
8. **Accessibility & mobile-first**, strong search/IA, prominent search bar.

## Recommendations for RVCE IEM site (implemented in the build)
- Adopt the page set: Home, About, Faculty, Research, Curriculum, Placements, Alumni, Events, FAQ, Contact.
- Lead Home with the IEM value proposition + headline stats (50 intake, >70% placements, ₹14.95 LPA, NBA-accredited).
- Add a dedicated **"What can an IEM graduate do?"** careers section (differentiator most peer sites do well).
- Build a **faculty research grid** linking IRINS/Scholar.
- Add **FAQ with structured data** to reclaim search traffic from aggregators.
